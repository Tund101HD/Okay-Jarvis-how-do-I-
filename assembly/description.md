> *"Sir, you are writing instructions the CPU will execute literally. Please type carefully."* — J.A.R.V.I.S. (probably)

This directory collects guides centered around **assembly languages** — mostly **NASM** and friends — covering toolchain setup, IDE integration, and the small rituals that turn "why is this just printing nothing" into a working binary.

If you're here, you're likely trying to:

- Get an IDE (CLion, VS Code, etc.) to assemble, link, and debug ASM without hand-rolling a build script.
- Learn assembly by actually running it, not by squinting at PDFs.
- Understand what your compiler has been quietly doing on your behalf for years.

## Guides in this directory

See the [main README](../README.md) for the indexed table. ASM-tagged guides live here and are linked from there.
