This directory collects guides centered around **Minecraft mod development** — the kind that runs alongside the game itself, on **Forge**, **Fabric**, **NeoForge**, or whichever loader is in fashion this week. Expect coverage of project bootstrapping, mappings, mixins, asset/data registration, and the IDE plumbing to make all of that bearable.

If you're here, you're likely trying to:

- Spin up a fresh mod project without battling Gradle for an afternoon.
- Understand the difference between Forge, Fabric, and NeoForge well enough to pick one.
- Add custom blocks, items, entities, or behaviors without breaking the registry.
