> *"Sir, the server is online. Try not to grief it."* — J.A.R.V.I.S. (probably)

This directory collects guides centered around **Minecraft plugin development** — the server-side flavor that runs on **Bukkit**, **Spigot**, **Paper**, and their many forks. Expect coverage of project bootstrapping, event listeners, commands, configuration, and the IDE setup that makes hot-reloading a plugin feel less like punishment.

If you're here, you're likely trying to:

- Stand up a Paper/Spigot dev server and connect your IDE to it.
- Write commands, listeners, and config files without copy-pasting the same boilerplate every time.
- Figure out which APIs actually exist on which fork, and which ones quietly got deprecated three versions ago.
